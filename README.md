# Lab

Personal research repository for machine learning, generative models, time-series, and cognitive / brain-inspired studies.

This repository is used for:

* Paper reproduction
* Model experiments
* Algorithm research
* Brain / psychology inspired ML
* Future on-device / NPU / neuromorphic work

---

## Structure

```
lab/
  diffusion/        # diffusion models, flow matching, generative models
  llm/              # LLM, transformers, language models
  time-series/      # forecasting, sequence models, TS foundation models
  psychology/       # cognition, neuroscience, brain-inspired ideas

  data/             # datasets (NOT tracked by git)
  outputs/          # experiment outputs / logs (NOT tracked)
  checkpoints/      # model weights / ckpt (NOT tracked)
```

---

## Storage policy

Large files must NOT be committed to GitHub.

Do not push:

* datasets
* model checkpoints
* logs
* tensorboard runs
* large numpy / torch files

These directories are ignored:

```
data/
outputs/
checkpoints/
```

Use local storage or external storage instead.
